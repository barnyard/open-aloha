Building Fitnesse
-----------------

Dependencies
------------

Apache Ant 1.7.0 or above
Java JDK 1.5 or above
Fitnesse installation

How to build
------------

1. create a properties file in the properties directory that refects your username and edit it appropriately.
2. Ensure Fitnesse is running
3. type ant

A build report will be found in build/fitnesse/fitnesse.html
